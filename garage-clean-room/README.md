# Garage Clean Room

Interactive project guide for converting the back section of the garage into a sealed, clean workspace / man cave.

## What's in here

- `index.html` — Full interactive guide with build phases, ASCII diagrams, Bunnings shopping list (with checkboxes), water management notes, and maintenance schedule

## Project summary

| Detail | Value |
|--------|-------|
| **Estimated cost** | $350–500 |
| **Time** | 1–2 weekends |
| **Enclosed area** | ~3.5m × 3m |
| **Skill level** | DIY / beginner |

### Build phases

1. **Prep & wall sealing** — Clear out, patch cracks, silicone all joints
2. **Raised floor system** — Timber bearers + plywood subfloor (water flows underneath to side door)
3. **Ceiling enclosure** — 200μ plastic sheeting on timber beams with battens
4. **Walls & entry** — Plastic walls + PVC strip curtain doorway
5. **Air management** — Box fan + furnace filter for positive pressure
6. **Man cave finishing** — Lighting, heating, TV, carpet, storage

### Key design decisions

- Raised floor lets water pass underneath and exit via the side door
- Side door stays **outside** the enclosure as drainage + ventilation
- Existing timber beams and steel studs used as frame — no freestanding structure
- Soundcheck gyprock back wall (shared with studio) already waterproofed
