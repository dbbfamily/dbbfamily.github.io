Install Sequence

Do this AFTER paper bags are down, BEFORE sugar cane goes on. ~45 minutes total.

1
Screw WiFi timer onto tap Use the included tap adaptor. Download Holman Home app first — have it ready to pair immediately.
Tap is right next to bed — no long runs needed
2
Run 13mm poly pipe along bed edge From timer along the retaining wall cap edge. This is your main supply line. Leave extra length at the end — capped for now, extend later to next bed.
3
Tee off poly pipe into bed Use a 13mm tee fitting to branch the Techline drip tube off the main poly pipe at the start of the bed.
4
Snake Techline drip tube through bed Lay it in S-curves across the paper bags. Bring it close to each tree zone — within 150mm of each trunk. Space the runs about 400-500mm apart to cover the full bed width.
Keep 50mm clear of each trunk itself
5
End cap the Techline Fold and clip the end, or use a proper end cap fitting. Make sure it's secure — a loose end will spray water everywhere.
6
End cap the poly pipe main line Cap the far end of the main supply line for now. When you expand to the next bed, remove the cap, insert a tee, extend.
This is how you expand in phases — no digging up existing work
7
Turn on and test before mulching Run the system manually from the Holman app. Walk the bed and check every dripper is flowing. Fix any issues now before the mulch buries everything.
8
Mulch over the top Sugar cane then pine bark go on top of the drip tube. It's now hidden, protected from UV, and delivering water directly to root zone.
9
Set timer schedule in Holman app 5:00am daily, 20-30 minutes per zone. Canberra summers — run daily. Once mulch settles and cooler weather arrives, reduce to every 2-3 days.
Enable rain skip if you add the moisture sensor
